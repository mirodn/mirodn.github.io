Project 3a


WP 1: Design your own application domain

Draw your personalized conceptual data model



WP 2: Design your relational database

Design a relational model implementing your application domain. Specify the tables, their columns, primary and foreign keys.

WP 3: Test Data

Use Mockaroo to generate random (but meaningful) test data. Make sure you have (at least) 100 instances including attribute values for each entity type and (at least) 100 “connections” for the relationships.

WP 4: Query

Specify a query that meets the following requirements.

  • It returns values of the A.1 attribute declared by A.
  • It returns the values of the A.1 attribute of A instances that are associated to B instances by means of the ab relationship.
  • The B instances they are connected to have a given value for the attribute B.2. This given value is an input to the query.
