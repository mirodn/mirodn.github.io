Project 2

Project 2 is independent from the other projects. My task was to implement a quiz application for food.


Application Behavior

  1. The user enters his or her name and selects a food property from a given list. The list must contain five properties listed in the dataset. Example list: fat, sugar, protein, vitamin C and iron. You can use a static list of properties. There is no need to query the data base.
  2. The application queries the database on Atlas for all foods having a non-zero value for the selected property. The application randomly selects one food (food1).
  3. The application queries the database on Atlas for two more foods (food2, food3). These foods must meet the following conditions:
    • The composition of food2 and food3 is measured in the same unit as food1. There are two distinct units2: gram (g) and milliliter (ml).
    • The value of the selected property must be non-zero and at least 50% lower than the value of food1.
  4. The application prints the names of food1, food2 and food3 in random order. The user is asked to identify the food with the highest value of the selected property. Example: if the user selected sugar in step 1, the user must identify the food containing the most sugar.
  5. The application checks the answer, states whether the answer was correct or not and lists the property value of each food. A correct answer is rated with one point, a wrong answer with zero points.
  6. Steps 2 to 6 are repeated 5 times in total. Points are summed up.
  7. The application saves the ranking as a document into a separate collection on Atlas. The document must contain at least the username, the selected property, the total points gained, and the time needed.
  8. The application lists the three best rankings. Rankings are first sorted by points and second by time.
